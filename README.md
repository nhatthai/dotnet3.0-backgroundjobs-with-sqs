# DotNet Background Jobs SQS
Read messages from Amazon SQS and then run Background Job

## Prerequisite
+ Asp Net Core 3.0
+ Entity Framework
+ Docker & Docker Compose
+ Amazon SQS
+ Hangfire for running Jobs
